# Media Assets

Public storage for image and video assets.

## Direct URLs

After pushing a file, use this format:

```text
https://raw.githubusercontent.com/TTTiant/media-assets/main/images/your-image.png
https://raw.githubusercontent.com/TTTiant/media-assets/main/videos/your-video.mp4
```

Put images in `images/` and videos in `videos/`.
